Visión general

Angular Rich Text Editor es un editor HTML WYSIWYG con muchas funciones y un editor de Markdown WYSIWYG. El editor de texto enriquecido se usa ampliamente para crear blogs, publicaciones en foros, secciones de notas, tickets de soporte (incidentes), secciones de comentarios, aplicaciones de mensajería y más. El control proporciona una interfaz de usuario eficiente para una mejor experiencia de edición con soporte móvil. Tiene una variedad de herramientas para editar y formatear contenido enriquecido y devuelve marcado HTML válido o contenido Markdown (MD). Permite a los usuarios insertar imágenes, enlaces, tablas y listas con arquitectura modular.

Editor HTML WYSIWYG

El componente Angular Rich Text Editor se puede utilizar como editor HTML WYSIWYG. Permite a los usuarios ver y editar contenido como marcado HTML y ver la fuente HTML usando el Code Viewicono.

Editor de rebajas

El componente Angular Rich Text Editor se puede utilizar como Angular WYSIWYG Markdown Editor. Permite que el contenido esté en formato de rebajas. Se puede obtener una vista previa del contenido de la sintaxis de rebajas mecanografiadas utilizando un complemento de terceros.

  • Obtenga más información sobre nuestro editor de Markdown

Modo en línea

Hace que el editor HTML en línea se muestre cuando hace clic o selecciona el contenido en un área editable.

  • Ejemplo de editor HTML en línea WYSIWYG angular

Modo iframe del editor HTML WYSIWYG

El editor de texto enriquecido angular admite la edición de iframe con atributos de iframe personalizados. También cambia fácilmente la apariencia del contenido del editor de Iframe cuando se utilizan archivos de script y CSS externos.

  • Documentación de edición de iframe

Barra de herramientas del editor HTML

The WYSIWYG HTML editor toolbar provides a variety of commands for editing and formatting the content. You can format text, paragraphs, insert images, insert hyperlinks, insert tables, insert lists, etc. The tool commands are grouped together based on related functionality.

  • HTML editor Toolbar example

Multi-row toolbar

The overflow toolbar commands are wrapped in the following rows within the toolbar.

Expand toolbar

The overflow toolbar commands hidden in the next row become visible when the toolbar is expanded via the expand icon.

Quick toolbar

This toolbar contains frequently used commands related to text, image, tables, and links. It helps access and format these elements (text, image, tables, links) quickly, and allows you to customize it.

  • Quick toolbar documentation

Floating the toolbar

The HTML WYSIWYG editor toolbar does not stick to the top of the editor, but floats within the editor area while scrolling the editor page.

  • Floating toolbar documentation

Fully customizable

The toolbar of the Angular Rich Text Editor component is fully customizable with its buttons. You can add or remove buttons and also group commands based on category. This helps you to design the toolbar based on the application needs like HTML blog editor, HTML email composer, and more.

Custom tool

Users can add their own commands(tools) with actions to the toolbar along with built-in commands.

  • Custom tools documentation

Editing experience

The Angular Rich Text Editor component provides a wide range of tools and options for a better editing experience.

Formatting text

You can apply formatting to a selected content, whole paragraphs, or specific words or characters:

  • Apply formatting to headings, quotations, code, paragraphs, etc.
  • Bold, italicize, underline, strikethrough, or capitalize letters and words.
  • Change paragraph indentation and spacing.
  • Select fonts and its size value.
  • Use subscripts and superscripts.
  • Select font color and the background color for the content.

Insert images

You can insert images in your content from a local path or server path with image captions and then link URLs to the images. You can upload images to a server with drag-and-drop or file upload. Additionally, the editor offers options to insert images in BLOB or Base64 formats.

Insert hyperlinks

You can insert or change the hyperlinks with customization options such as display text, tooltip, and link to be opened in a new window.

  • Learn about Insert hyperlink

Insert tables

You can insert and edit HTML tables to display grid-like tabular data in WYSIWYG Rich Text Editor. Allows manipulation of the table and its elements dynamically such as insert/remove rows and columns, resize whole table or rows or columns, and remove a table.

  • Insert HTML tables examples

Text alignment

The built-in settings are available to align your content to the left, center, right, or justify it.

Adding lists

Create ordered (numbered) or unordered (bulleted) lists to organize the content.

Full screen text editor

The built-in option makes the WYSIWYG HTML editor full screen size (full page editor) for editing the content.

Cut, copy, and paste

The rich text editor typically allows users to use clipboard operations (cut, copy, and paste).

Undo redo manager

The Angular Rich Text Editor manages the undo action history to store users’ editing actions, and also to customize the undo steps.

  • Learn more about Undo redo manager

Source code (HTML) view

Examine and edit the HTML code directly in source code and view the preview result (HTML live editor).

Paste from Microsoft Word

The Rich Text Editor offers built-in options to paste content with images from Microsoft Word and Microsoft Outlook to the editor by filtering out tags, attributes, and styles. Additionally, you can control if the content pastes as plain text, cleaned styles, or source-formatting styles.

Integrate with third-party libraries

Easy to integrate third-party libraries like spell checker, code mirror, etc., with the RTE to improve the content in Angular HTML text editor.

  • Codemirror integration example
  • Third-party integration documentation

Print the text editor content

Print the WYSIWYG Rich Text Editor content either by using the print option from the toolbar or programmatically with applied format style.

Auto hyperlink of URL

Configured to turn text into a hyperlink automatically when user types the URL as a plain text.

Angular form validations

The Angular Rich Text Editor seamlessly supports angular template-driven forms and reactive forms. The two types of validation are,

  • Required validation

  • Maximum length validation

  • Template-driven form example
  • Reactive form example

Limit content length

Users can restrict the content’s maximum number of characters.

  • Maximum character count limit documentation

Resizable editor

The Angular Rich Text Editor has built-in resizable support that helps widen or shrink the content area.

Touch-friendly and responsive WYSIWYG editor

The Angular Rich Text Editor (mobile HTML editor) recognizes touch gestures, allowing the user to swipe left or right to move the toolbar. The responsive, mobile friendly design provides the best user experience on all mobiles, tablets, and desktop form factors.

Cross-browser support

The Angular Rich Text Editor supports cross-browsers and cross-platforms. It also supports ie11 with the help of polyfills.

  • Learn more about our cross-browser compatibility

Globalization

Localization

You can customize all static texts used in the Rich Text Editor component using the localization library.

  • Change HTML editor language documentation

Right to Left (RTL)

The Angular Rich Text Editor supports right-to-left rendering, allowing the text direction and layout of the text editor to be displayed from right to left. This improves the user experience and accessibility for RTL languages such as Arabic, Farsi, Hebrew, etc.

  • Right to Left documentation

HTML editor accessibility

  • Fully supports WAI-ARIA accessibility that helps the editor be accessed by on-screen readers and assistive devices.
  • The UI element visuals such as foreground color, background color, line spacing, text, and images are designed based on the WCAG 2.0 standards.
  • Follows WAI-ARIA best practices for implementing all features using accessible keyboard shortcuts.
  • Learn about WYSIWYG HTML editor keyboard shortcuts

Developer-friendly APIs

There are sufficient client-side APIs available to optimize the process of editing content.

  • Rich Text Editor APIs documentation

With continuous improvement in Angular versions, the Angular Rich Text Editor is kept up to date to make it compatible with versions starting from 4 to the latest version.
